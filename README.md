# Atlantis

## Visão Geral do Projeto

O Atlantis é um sistema desenvolvido para gestão de parques aquáticos, clubes, hotéis e resorts, criado com o objetivo de oferecer uma solução moderna, flexível e acessível para o mercado. O projeto evoluiu a partir de um MVP (Minimum Viable Product) CLI até uma aplicação web completa, incorporando padrões de projeto, princípios de engenharia de software e integração com banco de dados.

Durante o desenvolvimento, o sistema passou por várias etapas, cada uma focada em aprimorar funcionalidades, arquitetura e experiência do usuário, sempre considerando feedbacks reais do mercado.

## Branches do Projeto

- **atv-i**  
  Primeira versão do projeto, focada no cadastro de clientes e dependentes, aplicando o padrão de projeto Protótipo.

- **atv-ii**  
  Evolução do sistema CLI, com aplicação dos padrões Singleton e Strategy, além de princípios SOLID. Inclui CRUD de clientes e dependentes.

- **atv-iii**  
  Adiciona o módulo de acomodações, utilizando o padrão Builder para criar diferentes tipos de quartos e implementando o controle de hospedagens.

- **atv-iv**  
  Protótipo navegável da interface gráfica, desenvolvido como SPA (Single Page Application), contemplando todas as funcionalidades do sistema CLI.

- **atv-v**  
  Versão web completa, baseada no protótipo SPA, com integração a banco de dados para armazenamento persistente das informações.

Cada branch representa uma etapa do desenvolvimento, permitindo acompanhar a evolução do projeto e facilitando a manutenção e futuras melhorias.
